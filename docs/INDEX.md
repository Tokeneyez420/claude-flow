## üéØ Quick Reference

### Essential Commands

```bash
# Installation
npx claude-flow@alpha init --force

# Basic Usage
npx claude-flow@alpha swarm "build REST API"
npx claude-flow@alpha hive-mind spawn "project"
npx claude-flow@alpha sparc tdd "feature"

# Development
npm install
npm run dev
npm test
npm run build
```
