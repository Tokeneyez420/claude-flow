# Hello World Application

A simple Python program that prints "Hello, World!" to the console.

## Requirements

- Python 3.x

## How to Run

1. Navigate to the hello directory:
   ```bash
   cd hello
   ```

2. Run the program:
   ```bash
   python hello.py
   ```
   
   Or make it executable and run directly:
   ```bash
   chmod +x hello.py
   ./hello.py
   ```

## Expected Output

```
Hello, World!
```
